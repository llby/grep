grep
======

これは継続的なgrepのためのツールです。  
FuelPHPのmodulesとして作成しています。  
modules部分しか公開していないため、そのまま動かすと、  

    \Auth::get_user_id();

といった部分で、エラーになるかもしれません。  
適当に組み込むか、  

    array(0,0);

などの、固定の値に変更してください。  
不明点があれば、ご連絡ください。
